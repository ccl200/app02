# app02
任务发布系统

1.安装软件
    安装python2.7
    安装wxpython

2.使用方法
    server是教师端
         配置server 下的 config.ini文件
         修改host为教师端IP地址
         修改server_store_file_path学生文件收集上来后的存储路径
         修改server_send_file_path学生文件下发的存储路径
         修改client_store_file_path教师下发文件的存储路径
         修改client_send_file_path学生发送文件的路径目录



    clinet是学生端
        配置client 下的 config.ini文件
        修改host为教师端IP地址
